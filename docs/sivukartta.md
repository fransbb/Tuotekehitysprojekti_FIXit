# FixIT Mobile – Sivukartta

1. Etusivu
   - Arvolupaus
   - Yleisimmät palvelut (näytönvaihto, akkuhuolto, ohjelmistot)
   - Luottamussignaalit (esim. “2000+ korjausta”)
   - CTA: Ota yhteyttä / Aloita huolto

2. Palvelut ja hinnat
   - Puhelimen huolto
   - Tabletin huolto
   - Tietokoneen huolto
   - Suuntaa-antava hinnasto
   - Huomio: tarkka hinta diagnoosin jälkeen

3. Näin korjaus etenee
   - Vaihe 1: Tuonti
   - Vaihe 2: Diagnoosi
   - Vaihe 3: Korjaus
   - Vaihe 4: Nouto

4. UKK (Usein kysytyt kysymykset)
   - Takuu
   - Aika-arviot
   - Maksutavat
   - Voiko tuoda ilman ajanvarausta

5. Yhteystiedot
   - Sijainti + kartta
   - Aukioloajat
   - Puhelinnumero ja sähköposti
   - Yhteydenottolomake (ulkoasu riittää)
